## Trip Overview

- **Duration:** 15 Days
- **Travelers:** 5 people
- **Budget:** ₹2-2.5 Lakh per person
- **Countries:** Czech Republic, Germany, Austria, Slovakia, Hungary

## The Route

```
Prague → Bohemian Switzerland → Dresden → Český Krumlov →
Hallstatt/Salzburg → Vienna → Bratislava → Budapest → Prague
```
